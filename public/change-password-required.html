Wajib Ganti Password

Untuk keamanan, password default harus diganti sebelum kamu bisa masuk dashboard.

⚠️ Akun kamu masih pakai password default. Ganti dulu sekarang.
Password harus memenuhi:
  • Minimal 10 karakter
  • Mengandung minimal 3 dari: huruf kecil, huruf besar, angka, simbol
  • Tidak mengandung username
  • Password baru = ulangi password
Keluar dan login dengan akun lain
