Cara Deploy ke VPS (5 Langkah)

1

Clone repo dari GitHub kamu

git clone https://github.com/USERNAME/self-hosted-dev-platform
cd self-hosted-dev-platform
2

Jalankan script setup otomatis

Script ini install Docker, Docker Compose, dan Nginx secara otomatis

chmod +x scripts/setup.sh && sudo bash scripts/setup.sh
3

Isi file environment

cp .env.example .env
nano .env   # Ganti domain, password, dll
4

Jalankan semua service

docker compose up -d
5

Tambah user (tiap anggota tim)

sudo bash scripts/add-user.sh namauser

Tiap user dapat VS Code di browser sendiri di subdomain mereka
